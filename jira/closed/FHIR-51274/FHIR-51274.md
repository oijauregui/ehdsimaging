# FHIR-51274: Update XtEHR mapping to current/released XtEHR datamodel

## Description

The specification mapping to the XtEHR datamodel is based on a snapshot from early April.

Since then the XtEHR has evolved. The XtEHR mapping in the specification needs to be  updated to a more current version of the XtEHR model. Preferably a released version.

## Resolution

- Status: Applied
- Resolution: Persuasive

We will update the mapping.

Preferably the released version will be used (expected late July), if none is published, a new snapshot should be taken around Aug 1.

Expectation is that the mapping will change but that it will not require substantial changes to the model.
