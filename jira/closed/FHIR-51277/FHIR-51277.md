# FHIR-51277: HL7 II: The recommendation section in the ImComposition should allow referral to ServiceRequest

## Description

Source: discussions with IHE-IDR and HL7-II during Madrid WGM

Align with IHE-IDR and allow referral from the recommendation section to ServiceRequest directly, including possibly a profile for a recommended order (next to CarePlan). This should shortly elaborate on the difference between order, plan, proposal or option. Also add instructions on when ServiceRequest and CarePlan should be used. Possibly allow for RequestOrchestration as well to support order sets.
