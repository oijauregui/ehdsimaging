# FHIR-51124: Move some of the mapping to the base/core profiles

## Description

Some of the generic mappings specified in the mapping page – e.g. address - should be moved to the base /core profile.

This applies for the models already covered by that guide.

Add a link pointing to the associated page in the base/core profile
