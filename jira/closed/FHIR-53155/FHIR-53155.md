# FHIR-53155: AdverseEvents and AllergyInterllerance events are not part of the document tree

## Description

When something goes wrong during an imaging procedure this is documented in AdverseEvent and AllergyIntollerance resources. There is no reference to these resources in the tree starting from the Composition or Diagnosticreport. As a result they are not included in the Document bundle.

Suggest to add a reference to these resources in the procedure section of Composition and DiagnosticReport.
