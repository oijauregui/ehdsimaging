# FHIR-51329: Documentation of additional information on medication administration

## Description

Procedure profile:  https://build.fhir.org/ig/hl7-eu/imaging/StructureDefinition-ImProcedure.html

MedicationAdministration: https://build.fhir.org/ig/hl7-eu/imaging/StructureDefinition-Report-EuMedicationAdministration.html

We cannot find a reference (except partOf element has R5:MedAdm) to "MedicationAdministration," where additional information on medication administration (especially dosage) could be recorded. In our opinion, "slicing" for "used" would be most suitable OR composition.order section can have a slice. But the  procedure.used element slice only lists "medications," "devices," etc.

## Resolution

- Status: Applied
- Resolution: Persuasive

To achieve the correct representation of the EHDSImagingReport.body.examinationReport.medication, a new ImMedication administration will be introduced, having a slice in .partOf that references ImProcedure.

And the mapping will be changed, pointing that logical model data point with the new ImMedicationAdministration profile, instead of Medication, as it currently exists.
