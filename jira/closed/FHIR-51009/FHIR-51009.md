# FHIR-51009: improve circular definition

## Description

The 'Imaging Manifest Reports Provider/Creator' is defined as "Provides the Imaging study manifest." - that's a circular definition, suggest to use alternative wording for 'manifest' in its definition.

## Metadata

- Issue Type: Change Request

- Assignee: Unassigned

## Resolution

- Status: Applied
- Resolution: Persuasive

Change the text 

bq. Provides the Imaging study manifest.

by 

bq. Responsible for assembling and making available a structured representation of imaging study metadata including references to imaging instances and related clinical context
