## Resolution

- Status: Applied
- Resolution: Persuasive with Modification

We will add additional actors in line with the ones defined by IHE-MHD.

This specification is about content and not workflow so no transactions will be defined.

We will add a note to the specification that we plan to add actor specific obligations in future releases based on feedback on that topic from XtEHR.
