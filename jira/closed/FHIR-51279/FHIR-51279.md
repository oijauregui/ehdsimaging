# FHIR-51279: HL7 II: Revise radiation dose approach

## Description

Source: discussions with IHE-IDR and HL7-II during Madrid WGM

Radiation dose is useful information that is difficult to interpret without taking the full context into account. The current specification has a radiation dose observation that contains a subset of the information present in the DICOM-SR document stored in the PACS. This observation does not add value on its own without taking the other information into account.

I suggest to remove the radiation dose observation components, update the mapping to refer to the DICOM source information. So only the qualitative information remains, refer to the Radiation-dose IG for more information. Optionally include an ImagingSelection instance to refer to the location of the DICOM-SR document.
