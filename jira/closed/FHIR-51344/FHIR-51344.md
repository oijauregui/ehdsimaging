# FHIR-51344: ImagingKeyImageDocumentReference.modality & Non-DICOMs

## Description

The introduction of the profile for Imaging Key Image Document Reference lists Non-DICOM images as on of the use cases. However, the profile contains modality as a mandatory element and has a ValueSet binding for a DICOM code list (CID33, which contains just DICOM related codes) on it.

Either relax modality to optional or provide a ValueSet that contains both DICOM and non-DICOM codes.
