## Resolution

- Status: Applied
- Resolution: Not Persuasive with Modification

The requested modality is coded in the ServiceRequest.code as the procedure to be performed.

We will create an example value set with the imaging procedure code and add this a a binding to ServiceRequestOrderEuImaging.code. We will state in the text that this represents the imaging examine that is requested.
