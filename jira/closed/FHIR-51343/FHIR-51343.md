# FHIR-51343: Datatype Profile Imaging SOP Class UID Identifier - not clear

## Description

The overall page for this datatype profile is not clear:

	the introduction states that "This profile on Identifier represents the SOP Class UID (0008,0018) for the Imaging Order."
	However the code 0008,0018 which is also used as a fixed value for the type is the one for the "SOP Instance UID"



Please clarify what this datatype profile should represent - the SOP Class UID or the SOP Instance UID.

## Resolution

- Status: Applied
- Resolution: Persuasive

The description of the profile is wrong. The identifier is representing a "SOP Instance UID".

We will change the description of the profile an check whether places where the profile is used are also using the correct description.
