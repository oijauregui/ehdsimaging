# FHIR-51284: Please add examples on how to use the specification

## Description

The proper way to use the specification is not clearly explained making the specification less accessible.

This should be improved by adding examples covering different aspects. Important aspects include:

	unstructured reports vs structured reports.
	the use of findings, impressions, key images
	different modalities
	different options present in the profiles



These examples will also help in checking the profiles.

Sections should be added to the narrative part of the specification that explains each example and leads the reader to what information goes where and why.
