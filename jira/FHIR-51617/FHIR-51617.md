# FHIR-51617: composition type binding

## Description

Currently composition type use hl7 code system HL7EuDocumentTypes (extensible) which currently include only one code imaging-report-v1-0.0 with definition ImagingManifest v 1.0.0. This does not reflect meaning of an imaging study (document).

I propose to consider use LOINC code 18748-4 Diagnosti imaging study. Alternatively we shoul also consider binding to the SNOMED CT code 4201000179104 Imaging report and/or its children.
