# FHIR-51175: Check consistency in the usage of patient/subject

## Description

(1) cardinality 

do you expect to have manifest and report without any subject ?

I suggest to set the cardinality to 1.. as it is used in the DocumentReference.

(1.b)

In this case update also the Bundle profile adding a patient slice 1..

 

(2) use of profiles

the usage of target profiles for the patient is not coherent across the guide in some case you have base resource (as in the report composition; ) in others the ImPatient or directly the PatinetEu profiles (see e.g. https://hl7.eu/fhir/imaging-r5/ConceptMap-EHDSPatientMap.html)
