## Description

The sole "structured" example is a cardiology stress echo (34 wall-motion Observations). There is no parallel radiology structured example (e.g., CT chest with per-organ findings, mammogram with BI-RADS category, thyroid with TI-RADS score).

The example set strongly shapes implementer expectations. A cardiology-only structured exemplar will nudge radiology vendors toward either over-implementing per-segment Observations or dismissing structured reporting altogether.

Proposed resolution:

Add a radiology structured example (e.g., chest CT by organ system, or mammogram with BI-RADS). Label the echo example as specialty-specific.
