# FHIR-51275: HL7 II: DiagnosticReport vs Composition

## Description

Source: discussions with IHE-IDR and HL7-II during Madrid WGM

DiagnosticReport in IHE-RAD.IDR is the main resource. The Composition is documentation of the content and should not be used as the principal access mechanism to the structure data. That should be the DiagnosticReport.

Ensure that all important sections/parts are represented on DiagnosticReport - use IDR's approach as guidance here.

Add/update rationale related to this approach to the document.
