# FHIR-56198: Where possible reuse terminology and structure definitions from MADO

## Description

The MADO specification duplicates some of the terminology in the imaging specification. Specifically:

	the accession number and study instance UID identifier definitions
	the requested procedure and imaging study references
	the anatomical region valueset
	document reference search parameter for bodysite and modality
