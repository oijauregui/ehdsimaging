# FHIR-56782: ValueSet for DocumentReference.status

## Description

The EEHRxF MHD DocumentReference Profile for Imaging Reports  (DocumentReferenceImagingReport) profile uses the IHE MHD ValueSet (DocumentReferenceStats), which does not include entered-in-error. 

It is necessary to remove " | entered-in-error" from the short description.

Furthermore, we have two questions:

	Why was entered-in-error removed?
	How was it possible to replace one required ValueSet (FHIR base) with another required ValueSet without generating an error?
