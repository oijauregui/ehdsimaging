# FHIR-51883: Remove locally defined EU profiles while awaiting for migration to EU Core

## Description

As agreed during a meeting held 20/8/2025 with Bvdh, Marc K and Rutt L, we'll drop the EU profiles defines in this IG and refactor the Im profiles having the bare FHIR profile as parent.

This is a temporary solution until these profiles are defined in EU core as proposed in FHIR-51123 , When that ticket is implemented  then the dependency of Im profiles will be reestablished with EU profiles.
