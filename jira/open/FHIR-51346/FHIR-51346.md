# FHIR-51346: Design Considerations, Image data access and endpoints

## Description

The design consideration lists the following options for image data access:
Endpoint profiles will be defined for at least the following image access mechanisms:

	DICOMweb WADO
	DIMSE
	Launch a DICOM viewer



However, the rest of the IG only talks about DICOMweb and the DICOM Viewer option.
Therefore, either remove the DIMSE option from this list or add further description on how it can be used throughout the IG.
