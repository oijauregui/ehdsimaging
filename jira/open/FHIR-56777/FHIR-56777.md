# FHIR-56777: Future: add a way to identify AI-generated or CAD-assisted findings

## Description

The IG doesn't provide a way to document that a finding was AI-generated or used computer-aided detection. §8 only says "AI results are stored as DICOM-SR documents"; §3 has no AI use case.

Since this is an increasingly common use case, there could be an explicitly listed use case or some way for the consumer to determine whether a finding is AI-generated/assisted vs human authored.

Proposed resolution:

For a future version of the IG: consider adding Device to performer targets and an AI-reading use case in §3.
