# FHIR-56734: partOf accepts MedicationAdministration as a target and is conformant; clarify whether the Imaging Report Provider must constrain to Procedure

## Description

partOf can accept a medication administration as a reference and be conformant — it's not clear that the reference needs to be a procedure, like would be expected for an imaging report.

Proposed resolution:

Clarify if the obligation is specific to the procedure reference when being provided by the Imaging Report Provider.
