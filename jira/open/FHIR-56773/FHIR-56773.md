## Resolution

- Status: Triaged
- Resolution: Persuasive with Modification

The reporter is correct in that the MS fields occur. The proposal is to change these MS labeled fields in obligations. This requires that the Xt-EHR mappings are updated as well.

We will add SHALL:able-to-populate obligations for DiagnosticReport: ImagingReport to:
 * extension[patientHistory]
 * extension[finding]
 * extension[communication]
 * result

Remove the MS indication.

These obligations will also be added to the Xt-EHR mapping table and spreadsheet.
