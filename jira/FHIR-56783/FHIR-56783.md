# FHIR-56783: Inclusion of IPS in the IHE FormatCode ValueSet used for imaging reports

## Description

We have observed that in the DocumentReferenceImagingReport profile, the .content.format element binds to the IHE FormatCode ValueSet (preferred). This ValueSet includes the code for the International Patient Summary (IPS).

We would like to ask whether it is appropriate for a ValueSet intended for imaging reports to contain an IPS code. Would it not be more suitable to restrict the ValueSet to format codes specifically related to imaging reports?
