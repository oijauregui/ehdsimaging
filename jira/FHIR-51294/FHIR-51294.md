# FHIR-51294: Clarify and further define use cases

## Description

The specification should add a section on use cases that show searching for and retrieval of reports and manifest, including variations of imaging viewing.

Based on these analysis it is likely that additional ActorDefinitions and CapabilityStatement with custom search parameters are required.
