Resource Profile: IM Image Viewer Endpoint

Official URL: http://hl7.eu/fhir/imaging-r5/StructureDefinition/ImImageIidViewerEndpoint Version: 0.1.0-ballot
Standards status: Draft Maturity Level: 1 Computable Name: ImImageIidViewerEndpoint

Copyright/Legal: Used by permission of HL7 Europe, all rights reserved Creative Commons License

This profile defines a placeholder for an Endpoint for a viewer that can be used to access the study, serie it is present on. The application is based on IHE-IID.

Usages:

  • Derived from this Profile: Manifest obligations for ImImageIidViewerEndpoint
  • Use this Profile: IM ImagingStudy Manifest
  • Refer to this Profile: IM Imaging Study
